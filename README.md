# Welcome to my portfolio! 

This project showcases my assignments for the 2026 UU GIS course as well as some mapping projects I have worked on in my own time. 

You can access the live version of the website below: 

**[Click Here to View the Live Website](https://alvarjl2807.github.io/GISPortfolio/index.html)**

This project includes different types of maps (choropleth maps, elevation maps, 3D models, and a mix of interactive and static maps).

Images can be found in the 'Media' folder.
Files for Qgis2threejs can be found in the 'data' and 'threejs' folders.
Javascript files can be found in the 'Javascript' folder.

If you want to load this website locally, you can clone the repository and open index.html.
